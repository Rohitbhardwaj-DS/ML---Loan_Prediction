📌 Project Overview

This project aims to build a Machine Learning model that predicts whether a loan application will be approved or rejected based on applicant details such as income, credit history, education, and employment status.

Loan approval is a critical decision for financial institutions. By automating this process using data-driven techniques, we can reduce risk, improve efficiency, and ensure faster decision-making.

🎯 Problem Statement

Given customer information, predict:
👉 Loan Status (Approved / Not Approved)

📂 Dataset

The dataset includes features like:

Gender
Marital Status
Education
Applicant Income
Coapplicant Income
Loan Amount
Credit History
Property Area

🔧 Data Preprocessing
Handled missing values
Converted categorical variables into numerical format
Feature selection and transformation

🤖 Machine Learning Models

The following models were applied:
Logistic Regression

Models were evaluated using:
Accuracy Score

📈 Key Insights
Credit History plays a major role in loan approval
Higher income increases approval chances
Certain demographic factors influence decisions
